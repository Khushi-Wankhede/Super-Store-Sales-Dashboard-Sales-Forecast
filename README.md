# Super-Store-Sales-Dashboard-Sales-Forecast
1. Project Title
📊 Super Store Sales Dashboard with 15-Day Sales Forecast

An interactive Power BI dashboard designed to analyze sales performance, profit trends, customer purchasing behavior, and forecast future sales using historical data.

2. Short Description / Purpose

The Super Store Sales Dashboard helps businesses monitor key sales KPIs, analyze regional and category-wise performance, and identify sales trends. It also includes a 15-day sales forecasting feature to support better business planning and decision-making.

3. Tech Stack

• 📊 Power BI Desktop – Dashboard development and visualization
• 📂 Power Query – Data cleaning and transformation
• 🧠 DAX – Measures, KPIs, and calculations
• 🔗 Data Modeling – Relationships and data integration
• 📁 File Format – .pbix

4. Data Source

The dashboard uses Super Store sales data containing:

• Orders and Sales Information
• Profit Data
• Customer Segments
• Product Categories and Sub-Categories
• Shipping Modes
• Regional and State-wise Sales Data
• Historical Sales Records for Forecasting

5. Features / Highlights
• Business Problem

Businesses often struggle to track sales performance, identify profitable products, and predict future demand using raw transactional data.

• Goal of the Dashboard

• Monitor sales and profit performance
• Analyze customer and regional trends
• Identify top-performing products and categories
• Forecast sales for the next 15 days
• Support data-driven business decisions
